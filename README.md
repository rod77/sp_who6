# SP_WHO6

## Permite verificar que procesos estan bloqueando.

### Si se ejecuta sin parametros viene ordenado por spid

```
sp_who6
```

### Se le puede agregar un parametro @orderByDate, para ordenar por fecha:

```
sp_who6 1
```
